# claudeの最適設定を研究するリポジトリ

## 作る/コマンド
/コマンドはルートディレクトリに該当のディレクトリがなかったら、自動的に生成されるようにする

/code-fix コード規約を読んで現在のブランチを修正する `/code-fix`　※オプショナルで指定ファイルのみ実行する `/code-fix xxx/xxx.tsx`
/markup スクリーンショットをマークアップする 画像がなかったらスクリーンショットがありませんと出力する `/markup xxx.png 対象ファイル`
/screenshots 変更差分をチェックして変更した部分のスクショを保存する機能 playwrightMCPを使う https://responsively.app/の導入を検討する `/screenshots`
/review ブランチ間の差分をレビューする `/review main feature/xxx`
/update 指定のブランチのバージョンをアップするコマンド `/update xxxx`
/summary PRのディスクリプションを作ってくれるコマンド マークダウン形式で吐き出す `/summary`
